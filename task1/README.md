Overview

The To-Do List Web App is designed to help users manage their tasks efficiently. It allows users to add, delete, edit, and mark tasks as complete, with data persistence via local storage.

Features
1) Add Tasks: Users can add new tasks using an input field and a button.

2)Delete Tasks: Tasks can be removed individually.

3)Mark as Complete: Tasks can be marked as complete, visually indicated by a line-through style.

4)Local Storage: Tasks are saved in the browserâ€™s local storage to persist across sessions.

Technologies Used

HTML: Provides the structure of the app.

CSS: Styles the app to ensure a pleasant user 
experience.

JavaScript: Adds interactivity and handles task management and local storage.

File Structure
index.html: Contains the HTML, CSS & JS file combined.

How to Run the App
Clone or download the project files.
Open index.html in a web browser.

Use the input field to add tasks, and the buttons to mark as complete or delete tasks.
